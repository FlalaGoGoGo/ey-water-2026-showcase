# LinkedIn Assets

Use these files as the starting point for the LinkedIn post package:

- `hero_desktop.png`
  - Best option for the main post image.
- `mobile_overview.png`
  - Good supporting image to show that the project page also works on phone layouts.
- `storyboard_desktop.png`
  - Use this as a crop source for a longer "evidence wall" style image.

Recommended posting order:

1. `hero_desktop.png`
2. a crop from `storyboard_desktop.png`
3. `mobile_overview.png`

If you want a cleaner second image later, the simplest next step is to crop the evidence/chart section out of `storyboard_desktop.png`.
